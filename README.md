A modern marketplace connecting dance students with teachers.

## Features
- **User Auth**: Sign up/Login as Student or Teacher.
- **Teacher Profiles**: Create and edit profiles with real photos (Supabase Storage).
- **Search**: Find teachers by style, name, or bio.
- **Responsive Design**: Mobile-first glassmorphism UI.

## Tech Stack
- Frontend: React (Vite)
- Backend/DB: Supabase (PostgreSQL + Auth + Storage)
- Hosting: Vercel

## Deployment
This project is connected to Vercel.
Environment Variables required:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
